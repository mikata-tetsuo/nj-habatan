# はば単トレーナー（はばトレ）

兵庫県教育委員会『はば単』2,500語対応の英単語学習アプリ。

## アプリを開く
https://mikata-tetsuo.github.io/nj-habatan/habatan_quiz.html

## 主な機能
- チャレンジモード（CEFR-J準拠・レベル別）
- テストモード（自由範囲設定）
- 単語帳モード（音声対応）
- 復習リスト
- ランキング（Googleログイン）
- ログインボーナス

## 単語データ
兵庫県教育委員会『はば単』
https://www2.hyogo-c.ed.jp/hpe/gimu/english/habatan
